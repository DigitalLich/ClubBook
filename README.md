# ClubBook

The project is being develop to be a management website for Book Clubs primarily for me and my victims/friends
that I was able to influence in following me into reading books with me.

Planned Features:
- Group Management
    - Group Name
    - Group Avatar
    - Group Description
    - Group Status
    - Group Members
    - Group Announcements
- Book Categorization
    - Fiction
    - Non-Fiction
- Reading Timeline
    - Sessions
    - Dates
    - History
- Forums (Maybe...)
